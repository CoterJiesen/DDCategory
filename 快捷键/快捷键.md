##组合用法1
1、cmd + shift + o 快速查找到文件

2、cmd +  shift + j 得到项目的位置

3、cmd + j 聚集到编辑器忠

4、ctrl + 6 输入函数名查找对应的函数

##组合用法2
1、cmd + \ (delete下面那个键)

2、cmd + 7 快速定位断点选择栏

3、cmd + y 使所有断点无效

4、option + 方向键定位到workSpace 按delete方式删除

##打开窗口
1、cmd + 1 定位文件

2、cmd + 回车   打开标准窗口
   
3、cmd + option + 回车 打开辅助窗口

4、cmd + option + 0 打开文档

##MAC快捷键

cmd + shift + n  创建一个文件夹 
cmd + delete 删除一个文件
cmd + shift + delete 删除垃圾桶

ctrl + shift + 最右上角的建，快速锁屏电脑
ctrl + cmd + space 调出表情。
